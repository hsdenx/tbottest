tbottest.powercontrolÂ¶

A module containing various additional connectors for controlling power. These are:

  • GpiopmControl - Power using a gpio pin

  • PowerShellScriptControl - Power using a shell script.

  • SispmControl - Power using sispmctl.

  • TinkerforgeControl - Power using DH Electronic TM-021 4-fach Relaismodul dh.

  • TM021Control - Power using ``_.
